# 🎨 CloudCanvas

> Build images like Photoshop. Create with AI like the future.

CloudCanvas is an AI-powered browser image editor built with **React** and **Cloudinary**.

I wanted to see how close I could get to building a Photoshop/Canva like editor powered entirely by Cloudinary. The result is CloudCanvas, where users can generate images with Cloudinary AI, upload their own media, and edit everything on a non-destructive, layer-based canvas. It includes features like Smart Expand, Object Removal, Background Removal, AI Restore, artistic filters, and optimized exports—all without leaving the browser.

> 🚀 Built for the **Cloudinary Creators Community June Mini Hack**
